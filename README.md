# Dotfiles with XDG Base Directory support
## Install

```
# printf "ZDOTDIR=~/.config/zsh\n" >> /etc/zsh/zshenv
$ git clone https://github.com/Su-Ky/dotfiles
$ cd dotfiles
$ ./install
```

Dotfiles uses [Dotbot](https://github.com/anishathalye/dotbot/) for installation.
